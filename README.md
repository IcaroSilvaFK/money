### Para rodar o projeto com todas as funcionalidades é nescessário <code>json-server</code>

### Clone o repositório e digite o seguinte comando no terminal

```shell
  yarn
```

### Para rodar

```shell
  yarn server
  // depois
  yarn dev
```
